# YBTagView

根据传进去的数组的数量，来判断应该显示几个tag，最多只支持4个tag。
当手指点击的点在四周边，即左右或上下留下的地方不足以显示完整时，会自动调整位置。
当拖动tagView超过图片显示的范围时，会自动调整到合适的位置。
